This program will encrypt and decrypt simple text files. Keep in mind due to the strength of the key (7470 bits) that files with multiple lines will potentially take 20-30 seconds to encrypt and decrypt.

For the following file selector, choose a text file you'd like to encrypt, then submit.

To decrypt a file, please upload the ciphered text in the first box, and the key in the second box, then submit.
